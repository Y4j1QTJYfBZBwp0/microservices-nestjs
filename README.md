## Description

Repository containing my code generated during training "Nest.js: criando uma API Restful"

## Tech Stack

* VSCode
* GitHub Copilot
* GitHub Copilot Labs
* Nestjs
* Jest
* TestContainers
* SuperTest
* TypeORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
